WORKSHOP REPOSITORY

Welkom bij de workshop repository.
In deze repository worden alle workshops en bijbehorende opdrachten per week bijgehouden. Iedere student commit hier wekelijks zijn of haar gemaakte opdrachten.

DOEL
Het doel van deze repository is:
Overzicht houden van alle workshops
Voortgang per week inzichtelijk maken
Leren werken met Git en versiebeheer
Professioneel structureren van code

STRUCTUUR
De repository is ingedeeld per week en per workshop.
Per week zijn er 4 workshops.
Zorg dat je jouw opdrachten altijd in de juiste week- en workshopmap plaatst.

Belangrijk:
Plaats geen bestanden buiten de bestaande mappen
Gebruik duidelijke bestandsnamen
Houd de structuur netjes en overzichtelijk

WERKWIJZE
Werk aan je workshop
Werk altijd in de juiste week- en workshopmap.
Voeg je wijzigingen toe
Gebruik het commando: git add .

Maak een duidelijke commit
Gebruik altijd een duidelijke commit message, bijvoorbeeld:
Week 3 - Workshop 1 afgerond
Week 2 - Workshop 4 verbeterd
Week 5 - Workshop 2 extra opdracht toegevoegd

Gebruik geen commit messages zoals:
update
test
klaar

Push je werk
Zorg dat je wijzigingen naar de repository worden gepusht met git push.

AFSPRAKEN
Commit minimaal na elke workshop
Zorg dat je code werkt voordat je commit
Houd je code netjes en leesbaar
